# FeedbackMessage

O **FeedbackMessage** fornece um bloco visual e textual padronizado, que ajude a guiar o usuário sobre o que está acontecendo e quais próximos passos podem ser tomados.

Use o **FeedbackMessage** para comunicar estados como listagens vazias, falhas (ex.: erros 400 ou 500) ou buscas sem resultados. O componente inclui campos para título e descrição explicativa, com opção de exibir o ícone e botão de ação conforme o contexto da aplicação.
