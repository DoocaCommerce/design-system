# Tokens primitivos

Tokens primitivos são aqueles tokens base para gerar qualquer outro token, 
constinuindo parte fundamental do Design System.  Esses tokens não podem ser modificados 
pelos usuários de modo a garantir a integridade de todos os outros tokens.
