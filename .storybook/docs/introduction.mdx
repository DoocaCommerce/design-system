# Commerce Suite

O Design System Commerce Suite oferece orientações e componentes para a criação de interfaces consistentes e escaláveis para sistemas categorizados como Backoffice no setor do comércio eletrônico.

Ele foi construído a partir de uma biblioteca de componentes em Vue.js 3 e está em constante evolução para atender às necessidades de nossos usuários e aos padrões e requisitos de um design system moderno.
